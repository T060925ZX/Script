## Ubuntu安装Node.js
支持Ubuntu20.4/Ubuntu22.04/Ubuntu24.04/Ubuntu24.10

用下面的命令来安装
```
sudo wget "https://raw.githubusercontent.com/T060925ZX/Script/refs/heads/main/yz-cli" -O /usr/local/bin/yz
```

国内网络问题可以使用以下镜像
```

```

安装完成后执行
```
sudo chmod +x /usr/local/bin/yz
```
